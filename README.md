# KAIROX V5

AI-powered financial news intelligence platform. Built in public.

## What this is

KAIROX ingests financial news from multiple sources, deduplicates articles
using semantic similarity, generates AI summaries, and surfaces insider trading
signals — giving retail investors the kind of signal layer previously only
available to institutional players.

## Stack

- **Backend:** Python + FastAPI
- **Database:** PostgreSQL + pgvector
- **AI:** HuggingFace embeddings + Grok API
- **Mobile:** React Native (Phase 3)
